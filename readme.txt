Easily migrate your Wordpress database from one web address to another.

== Description ==

Makes migrating your Wordpress website from one web address to another as simple as possible. This plugin enables you to export your MySQL database while replacing all occurences of the old web address with the new one.

Usage

Select' Bamboo Migration' from the 'Tools' Menu, enter the new web address of your website and click 'Export Database'. An sql file will be generated and made available for download. Import this sql file into the database for your new website. (NOTE: All existing data on the new website will be overwritten. Please ensure this is what you want).
